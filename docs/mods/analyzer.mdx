# Analyzer

This adds five commands primarily designed for [Bongo](https://modrinth.com/mod/bongo). Although the mod itself provided
a similar feature, this implementation significantly enhances it.

---

## Commands

| Command                  | Description                                                   |
|--------------------------|---------------------------------------------------------------|
| `/analyzer items`        | Prints all items in `config/analyzer/items.txt`               |
| `/analyzer entities`     | Prints all monsters/animals in `config/analyzer/entities.txt` |
| `/analyzer biomes`       | Prints all biomes in `config/analyzer/biomes.txt`             |
| `/analyzer advancements` | Prints all advancements in `config/analyzer/advancements.txt` |
| `/analyzer potions`      | Prints all potion effects in `config/analyzer/potions.txt`    |
