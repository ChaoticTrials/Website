# Analyzer

import ProjectHeader from '@site/src/components/ProjectHeader';

<ProjectHeader slug="analyzer" mcVersion="1.17" />

This mod adds five commands primarily designed for [Bongo](https://modrinth.com/mod/bongo). Although Bongo itself
provides a similar feature, this implementation significantly enhances it.

---

## Commands
| Command                  | Description                                                    |
|--------------------------|----------------------------------------------------------------|
| `/analyzer items`        | Outputs all items to `config/analyzer/items.txt`               |
| `/analyzer entities`     | Outputs all monsters/animals to `config/analyzer/entities.txt` |
| `/analyzer biomes`       | Outputs all biomes to `config/analyzer/biomes.txt`             |
| `/analyzer advancements` | Outputs all advancements to `config/analyzer/advancements.txt` |
| `/analyzer potions`      | Outputs all potion effects to `config/analyzer/potions.txt`    |
