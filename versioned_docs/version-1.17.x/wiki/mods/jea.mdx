# Just Enough Advancements

import ProjectHeader from '@site/src/components/ProjectHeader';

<ProjectHeader slug="jea" mcVersion="1.17" />

**Just Enough Advancements (JEA)** is an addon for [Just Enough Items (JEI)](https://modrinth.com/jei) that displays
advancements within JEI. This addon enhances your gameplay by allowing you to search for advancements like items, and
provides guides on what you need to do to reach them, even if they are not yet unlocked.

---

![](/img/projects/jea/animated.gif)

## Search Advancements

- **Advanced Search Capabilities:** The addon allows you to search for advancements within
  [Just Enough Items (JEI)](https://modrinth.com/jei) just like you would with items.

## Advancement Details

- **Guidance:** Each advancement displays the requirements to unlock it.
- **Detailed Descriptions:** Click on any advancement to view a detailed description, complete with animations, that
  explains what you need to do to unlock it.

## Client-Server Compatibility

- **Server Requirement:** Since the client is typically only aware of completed advancements, this mod must be installed
  on both the client and the server.
