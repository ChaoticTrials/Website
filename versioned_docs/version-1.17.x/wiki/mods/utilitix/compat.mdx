# Compatibility
## Jade 🔍
This mod is fully compatible with [Jade 🔍](https://modrinth.com/jade). When viewing
the [Experience Crystal](blocks.mdx#experience-crystal), Jade will display the current levels contained within the
crystal.

## Just Enough Items
This mod is also compatible with [Just Enough Items](https://modrinth.com/jei). Custom recipes, such as gilding,
will be visible through JEI.
